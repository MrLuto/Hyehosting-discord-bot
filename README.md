
# HyeHosting discord

an old discord bot for the company hyeservices (hyehosting)


## Installation

just clone install change settings and run it

```bash
  git clone https://github.com/MrLuto/Hyehosting-discord-bot.git
  cd Hyehosting-discord-bot
  npm install
```

```bash
  node index.js
```
